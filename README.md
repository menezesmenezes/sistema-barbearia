# Barbearia Organizada

Projeto separado para facilitar a apresentação, manutenção e edição do sistema.

---

## Estrutura do Projeto

* `index.html` — Estrutura principal da página
* `css/style.css` — Estilização e visual do sistema
* `js/config.js` — URL e chave do Supabase
* `js/database.js` — Funções de integração com banco de dados
* `js/state.js` — Variáveis globais e configurações gerais
* `js/utils.js` — Funções auxiliares
* `js/client.js` — Área do cliente e agendamentos
* `js/admin.js` — Painel administrativo
* `js/navigation.js` — Navegação e login
* `js/init.js` — Inicialização do sistema
* `sql/schema.sql` — Script SQL do banco de dados

---

## Como Executar

Abra o arquivo `index.html` utilizando a extensão Live Server no VS Code.
