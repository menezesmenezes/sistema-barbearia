# Barbearia Organizada

Projeto separado para ficar fácil de apresentar e editar.

## Estrutura

- `index.html` — estrutura da página
- `css/style.css` — todo o visual
- `js/config.js` — URL e chave do Supabase
- `js/database.js` — funções de banco de dados
- `js/state.js` — variáveis globais e logo
- `js/utils.js` — funções auxiliares
- `js/client.js` — tela do cliente/agendamento
- `js/admin.js` — painel administrativo
- `js/navigation.js` — navegação e login
- `js/init.js` — inicialização do sistema
- `sql/schema.sql` — script do banco

## Como rodar

Abra o `index.html` com Live Server no VS Code.
